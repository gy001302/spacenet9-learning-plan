# 经典教程总引导

这份文件不是要你一口气看完，而是告诉你：

- 哪些经典教程是用来补 **CV / 配准算法** 的
- 哪些经典教程是用来补 **PyTorch 最小工作流** 的
- 它们分别在整套 SpaceNet 源码理解路线里扮演什么角色

## CV / 配准算法优先看
1. OpenCV Geometric Transformations
2. OpenCV Feature Matcher Tutorial
3. OpenCV Feature Matching + Homography
4. scikit-image transform / robust matching examples

## PyTorch 优先看
1. Quickstart
2. Tensor tutorial
3. Dataset / DataLoader tutorial
4. Saving & Loading Models
5. Transfer Learning tutorial

## 使用原则
- 不求看全，只求看对位置
- 每学一个教程，都要问：
  1. 它补的是哪一块能力？
  2. 它最后会映射到哪类 SpaceNet 代码？
