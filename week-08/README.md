# Week 8: 研究提升周（可选加强）

**本周目标：** 开始从“能读懂”迈向“能比较、能改造、能写总结”。

## 本周建议节奏

- 每天 2 小时
- 先抓主线，不逐行抠代码
- 每天结束要有书面输出（3~5 条笔记或一张流程图）

## 每日主题

- Day 1: 重读 SpaceNet9 paper/context — 把工程理解对回论文语境。
- Day 2: 查遥感配准相关综述 — 形成 broader context。
- Day 3: 补非刚性配准/薄板样条 — 把插值和几何关系讲清。
- Day 4: 补跨模态匹配论文阅读 — 挑 LoFTR / LightGlue / RoMa 各读摘要和图。
- Day 5: 写方案综述初稿 — 输出 2~3 页总结。
- Day 6: 修改与补洞 — 回头补最薄弱的一块。
- Day 7: 最终总结 — 完成个人知识图谱。
## 本周核心问题

- 你是否已经形成自己的 SpaceNet9 技术图谱？
- 你能否不用看代码，口头讲清 3 种主流方案？
- 下一阶段是复现、改造还是写总结？

## 建议最低完成线

- 至少完成：完成一份最终总结或综述草稿。
