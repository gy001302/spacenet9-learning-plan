# Week 6: 第1名：高度/深度先验与位移回归

**本周目标：** 理解为什么冠军方案不满足于纯几何拟合，而是把高程/深度引入位移建模。

## 本周建议节奏

- 每天 2 小时
- 先抓主线，不逐行抠代码
- 每天结束要有书面输出（3~5 条笔记或一张流程图）

## 每日主题

- Day 1: 读冠军方案主流程 — 抓重投影、匹配点、depth/elevation、回归、输出。
- Day 2: DEM / DSM / depth 对配准的影响 — 建立高度导致位移差异的直觉。
- Day 3: 回归而非纯变换 — 理解线性回归 + RANSACRegressor 在冠军方案里的角色。
- Day 4: 对比第1名与第2名 — 一个更偏“位移回归+先验”，一个更偏“系统级 pipeline”。
- Day 5: 对比第1/2/4名 — 形成自己的方案谱系。
- Day 6: 写个人 baseline 设计 — 用你自己的话写一个最小方案：matcher→RANSAC→dense field。
- Day 7: 总复盘 — 完成整套学习闭环：任务、技术谱系、自己的下一步复现方向。
## 本周核心问题

- 第1名、第2名、第4名的思路主线分别是什么？
- 你如果自己做 baseline，会选哪条路线？
- 你现在最薄弱的一块是 CV、PyTorch 还是遥感跨模态理解？

## 建议最低完成线

- 至少完成：写一页三大方案对比。
