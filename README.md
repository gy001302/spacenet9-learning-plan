# SpaceNet9 学习计划

> 适用对象：已有 GIS + Python 基础，每天约 2 小时学习时间。

## 使用方式

- 本计划按 **8 周** 设计，其中：
  - **前 6 周为主线必修**
  - **第 7-8 周为可选加强/复现与研究提升**
- 每周一个文件夹，每天一个子文件夹。
- 每个 day 文件夹内包含：
  - `plan.md`：当天学习目标、2 小时分配、任务清单、产出
  - `resources.md`：当天推荐学习资料（在线资源）
- 建议每天按以下节奏执行：
  - 40 分钟：补概念/理论
  - 50 分钟：读代码/做实验
  - 30 分钟：记笔记/画图/总结

## 目录导航
- week-01/：任务理解与配准主线
- week-02/：PyTorch 推理与现代 matcher
- week-03/：分割、Patch 推理与辅助先验
- week-04/：第2名主干：global/local registration
- week-05/：训练基础与可复现能力
- week-06/：第1名：高度/深度先验与位移回归
- week-07/：复现准备周（可选加强）
- week-08/：研究提升周（可选加强）


## 精修说明

- 本版为第二轮精修：补充了每周 `self-check.md`，用于周末自测。
- 每周 README 里增加了“本周核心问题”和“建议最低完成线”。
- 每天的计划文件保持 2 小时可执行，不追求把资料一次看完。
- 每天若时间不足，优先顺序固定为：**1 个核心资料 > 对应仓库文件 > 书面总结**。
